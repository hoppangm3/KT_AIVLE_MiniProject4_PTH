# 프로젝트 고도화 로드맵 (Roadmap)

다음은 프로젝트를 고도화하기 위한 기능 개선 태스크 리스트입니다.

- [ ] **1단계: UI/UX 및 디자인 고도화**
  - [ ] 다크 모드 (Dark/Light Theme) 토글 기능 추가
  - [ ] 프레이머 모션(Framer Motion)이나 CSS 트랜지션을 활용한 부드러운 페이지/컴포넌트 애니메이션
  - [ ] 데이터 로딩 중 스켈레톤(Skeleton) UI 적용
- [ ] **2단계: 핵심 기능 확장**
  - [ ] 도서 목록 검색 창 및 카테고리 필터링 기능 구현
  - [ ] 도서 리스트 무한 스크롤(Infinite Scroll) 또는 페이지네이션 적용
  - [ ] 상세 페이지 UI 업데이트 및 기능 추가 (피그마 디자인 반영 예정)
    - [ ] 사용자 댓글(리뷰) 작성 기능 및 UI 배치
    - [ ] 좋아요 / 싫어요 기능 추가
    - [ ] 파일 다운로드 버튼 추가
  - [ ] "나만의 서재" 탭 추가 (읽고 싶은 책, 읽고 있는 책, 다 읽은 책 분류 기능)
- [ ] **3단계: 상태 관리 및 프론트엔드 최적화**
  - [ ] React Query (TanStack Query) 도입을 통한 데이터 캐싱, 로딩, 에러 핸들링 일원화
- [ ] **4단계: 백엔드 및 인프라 개선 (심화)**
  - [ ] JSON-Server 대신 실제 DB (MongoDB / PostgreSQL) 및 Node.js 백엔드 서버 연동
  - [ ] 회원가입 / 로그인 (JWT 인증) 시스템 추가 및 유저별 도서 관리 기능 분리
